🔑 Stockify Licence Generator

Generate the SHA-256 hash for a customer’s unique licence key. This tool is for the seller only.

Instructions:

  1. Enter the desired licence key (a unique code for the customer).
  2. Click Generate Hash.
  3. Copy the resulting hash and paste it into config.js as the value for LICENCE_KEY_HASH.
  4. Deliver the original licence key (not the hash) to the customer. They will enter it in the activation screen.

Never share the hash publicly – it is used for validation only.
